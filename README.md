# Inyeon-Upstream
> "You are Daniel Craig but life owes you a Vesper Lynd? Let Inyeon be your Casino Royale powerful assistant."

Inyeon (인연) is a CLI tool providing a "fated connection" between you and your git workflow.

## Features
* **AI Diff Analysis:** Explains changes, refactors, and security implications.
* **Conventional Commits:** Auto-generates team-ready commit messages.
* **Privacy-First:** Your code stays on your infrastructure.
